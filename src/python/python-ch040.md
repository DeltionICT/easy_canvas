### Cesuur
* Toets: max 3 punten (=cijfer * 0.3)
* Opdracht 1 SFO: max 3 punten
* Opdracht 2 SFO: max 2 punten
* Opdracht 3 SFO: max 2 punt

Bijvoorbeeld:
* Toets = 7.2
* Opdracht 1 = 3pt
* Opdracht 2 = 1pt
* Opdracht 3 = 0pt

> (7.2 * 0.3) = 2.2 + 3 + 1 + 0 = 6.3 

of

> (3.5 * 0.3) = 1.05 + 2 + 1 + 0 = 4
